The Basic CSS Cheatsheet

Selectors

Name Description Example
* Selects ALL elements * {color: red;}
Element Selects specific elements from HTML p {color: red;}
Element, element, element Selects multiple elements from HTML with similar properties p, h1, h2 { color: red;}
.Class Selects elements with classes set in HTML class=red (HTML)
.red {color: red;}
#ID Selects elements with ID in HTML (CAN ONLY BE USED ONCE) id=red (HTML)
#red {color: red;}

Properties

Name Description Example
color Changes color of text p {color: red;}
background-color Changes background color of element p {background-color: red;}
border-style Specifies border size, color, and style p {border-style: 1px, red, dotted;}
margin Specifies space between other elements p {margin: 15px 10px 15px 10px;}
p {margin-top: 15px;}
padding Specifies space between content and border of an element p {padding: 15px 10px 15px 10px;}
p {padding-top: 15px;}
height/width Specifies the height and/or width of a container p {height: 10px;
width: 10px;}
text-align Aligns text horizontally inside it's container p {text-align: center;}
vertical-align Aligns text vertically inside it's container p {vertical-align: center;}
font-family Changes the font of an element. (don't forget to import special fonts into HTML or CSS) p {font-family: serif;}
